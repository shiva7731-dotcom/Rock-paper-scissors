# Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game developed using **Kotlin**. The application allows users to play against the computer, which makes random choices. The game determines the winner based on the classic Rock-Paper-Scissors rules and displays the result instantly.

## 🎮 Project Overview

The Rock Paper Scissors Game is a beginner-friendly project designed to demonstrate core Kotlin programming concepts such as conditional statements, random number generation, user input handling, and game logic implementation.

Players select either Rock, Paper, or Scissors, and the computer randomly chooses its move. The application then compares both choices and displays the game outcome.

## ✨ Features

* Play against the computer
* Random computer move generation
* Instant win, lose, or draw result
* Clean and easy-to-use interface
* Lightweight and efficient implementation
* Beginner-friendly game logic
